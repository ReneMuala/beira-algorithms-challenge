# Beira Algorithms Challenge

Este repositório contém informações sobre o desafio que **propus** durante o challenge de algoritmos da [**Oficina do Futuro**](https://oficinadofuturo.org.mz) na **Beira**.

## Problema

A Sra. Eliane tem um conjunto de anotações sobre eventos com datas específicas e deseja apresentá-las em ordem crescente pela data mais antiga mencionada em cada uma delas. Ela também gostaria de identificar a palavra mais repetida na última anotação (após a ordenação).

## Regras

1.  **Proibido o uso de IA**
2.  **Proibido realizar consultas externas**
3.  **Proibido comunicar-se com membros de outros grupos**
4.  **Proibida a utilização de bibliotecas externas**
5.  **Critérios de desempate/vitória:** Ganha o algoritmo que:
      * For mais rápido;
      * Tiver melhor complexidade temporal; ou
      * Usar menos recursos (memória/CPU).

# Uma solução para o desafio da OdF 2026

# problema

A Sra. Eliane tem um conjunto de anotações sobre eventos com datas específicas e ela deseja apresentá-las em ordem crescente pela data mais antiga mencionada em cada uma delas. Ela também gostaria de apresentar a palavra mais repetida na última anotação.

# Regras

1. É proibido o uso de IA
2. É proibido realizar consultas externas
3. É proibido comunicar-se com membros de outros grupos
4. É proibida a utilização de bibliotecas externas
5. Ganha o algoritmo que:
- For mais rápido, ou
- Tiver melhor complexidade temporal, ou
- Usar menos recursos.
